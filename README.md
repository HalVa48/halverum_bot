# VPN Telegram-бот

Этот бот предоставляет VPN-услуги на базе AmneziaVPN. Пользователи могут просматривать тарифы, производить оплату и получать конфигурации для подключения.

## Функционал

- Просмотр тарифных планов
- Оплата через платежную систему
- Автоматическая генерация конфигураций AmneziaVPN
- Управление подписками

## Установка

1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте конфигурацию в `config.py`
3. Запустите бота: `python main.py`
