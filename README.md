# VPN Telegram Bot

Telegram бот для управления VPN конфигами через API AmneziaVPN.

## Возможности

- 🔐 Авторизация по паролю
- ➕ Создание до 3 конфигов на пользователя  
- 📁 Получение конфигов в виде файлов (.conf)
- 📱 QR-коды для быстрого подключения
- ⏸️ Отключение/восстановление конфигов
- 🗑️ Удаление конфигов
- 📊 Статистика использования
- 💾 Автоматические бэкапы серверов (ежедневно в 03:00)

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Скопируйте конфигурацию:
```bash
cp .env.example .env
```

3. Отредактируйте `.env`:
```env
BOT_TOKEN=ваш_токен_бота
VPN_API_URL=http://localhost:8082
VPN_API_EMAIL=admin@amnez.ia
VPN_API_PASSWORD=admin123
ACCESS_PASSWORD=vpn2024
```

## Запуск

```bash
python bot.py
```

## Команды

- `/start` - начать работу с ботом
- `/backup` - ручной бэкап всех серверов (только для авторизованных пользователей)

## Автоматические бэкапы

Бэкапы создаются ежедневно в 03:00 по расписанию. Для изменения времени отредактируйте `bot/scheduler.py`.

## Структура проекта

```
vpn_bot/
├── bot.py              # Точка входа
├── requirements.txt    # Зависимости
├── .env.example       # Пример конфигурации
├── api/               # Клиент для VPN API
├── bot/               # Основной код бота
│   ├── config.py      # Конфигурация
│   ├── scheduler.py   # Планировщик бэкапов
│   ├── handlers/      # Обработчики сообщений
│   ├── keyboards/     # Клавиатуры
│   └── middlewares/   # Middleware
└── database/          # Модели и БД
```
